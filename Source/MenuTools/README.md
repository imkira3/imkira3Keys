# MenuTools
MenuTools is a simple program that extends the basic functionality of the [Windows system menu](http://en.wikipedia.org/wiki/Common_menus_in_Microsoft_Windows#System_menu).

## Main features
* Process priority
* Window transparency
* Always on top
* Minimize to tray
